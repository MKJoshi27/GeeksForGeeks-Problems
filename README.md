# GeeksForGeeks-Problems
It contain all the problems of GeeksforGeeks with my Solutions.

(1.) School - 0 Marks <br />
(2.) Basic - 1 Marks <br />
(3.) Easy - 2 Marks <br />
(4.) Medium - 4 Marks <br />
(5.) Hard - 8 Marks .
